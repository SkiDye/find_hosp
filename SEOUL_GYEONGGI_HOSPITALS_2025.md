## 🔌 API를 통한 데이터 수집 전략

### 우선순위별 수집 계획

#### Phase 1: 상급종합병원 (21개)
```
대상: 서울 14개 + 경기남부 7개
방법: API 종별코드 clCd=11 사용
예상 소요: 1회 API 호출로 전체 수집 가능
```

#### Phase 2: 일반 종합병원
```
대상: 서울·경기 종합병원 (약 100개 추정)
방법: API 종별코드 clCd=21 + 지역코드
수집 지역:
- 서울: sidoCd=110000
- 경기: sidoCd=410000
```

#### Phase 3: 일반 병원 및 의원
```
대상: 병원(clCd=31), 의원(clCd=41)
방법: 시군구별 분할 수집
주요 타겟:
- 강남구, 서초구, 송파구
- 분당구, 수지구, 영통구
```

### API 호출 예시

```javascript
// 서울 상급종합병원 전체 수집
const seoulTertiaryHospitals = await publicDataAPI.getHospitalList({
  sidoCd: '110000',  // 서울
  clCd: '11',        // 상급종합
  numOfRows: 100
});

// 경기도 수원시 종합병원 수집
const suwonGeneralHospitals = await publicDataAPI.getHospitalList({
  sidoCd: '410000',  // 경기
  sgguCd: '411003',  // 수원시
  clCd: '21',        // 종합병원
  numOfRows: 100
});

// 분당구 전체 병원/의원 수집
const bundangAllHospitals = await publicDataAPI.getHospitalList({
  sidoCd: '410000',  // 경기
  sgguCd: '411023',  // 분당구
  numOfRows: 100,
  pageNo: 1
});
```

### 2. 사용자 경험 최적화

**검색 시나리오:**
```
사용자 위치: 경기도 성남시 분당구
→ 추천 병원: 분당서울대병원 (상급종합)
→ 인근 병원: 서울아산(송파), 삼성서울(강남), 강남세브란스

사용자 위치: 경기도 이천시
→ 추천 병원: 경기도의료원 이천병원
→ 인근 상급종합: 분당서울대(40km), 아주대(50km)
```

### 3. 병원 관리자 등록 우선순위

**Phase 1: 상급종합병원 21개**

**Phase 2: 지역 거점 종합병원**
